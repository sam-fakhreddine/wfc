    direction TB

    [*] --> Queued
    Queued --> Scheduled : dispatcher assigns
    Scheduled --> Running : agent picks up
    Running --> Validating : execution complete
    Validating --> Succeeded : all checks pass
    Validating --> Failed : quality check fails
    Running --> Failed : runtime error
    Failed --> Queued : auto-retry (max 3)
    Failed --> Dead : retries exhausted
    Succeeded --> Archiving : retention policy
    Archiving --> Archived : compressed + stored
    Archived --> [*]
    Dead --> [*]
    Queued --> Cancelled : user cancels
    Running --> Cancelled : user cancels
    Cancelled --> [*]
